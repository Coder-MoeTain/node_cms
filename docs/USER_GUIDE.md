# User Guide (Public Site)

For visitors and content readers on the public NodePress website.

## Browsing content

- **Home** — featured posts, portal sections, or theme homepage
- **Blog** — list of published articles
- **Categories / Tags** — filter posts by topic
- **Search** — find posts and pages by keyword
- **Contact** — send a message to site administrators

## Language

If the site administrator enabled translations, use the language switcher in the header to view UI labels and (where configured) post content in another language.

## Comments

On supported posts you may:

1. Enter name, email, and comment text
2. Submit for moderation (comments appear after admin approval)

## Accounts

Some sites offer **Register** and **Login** for subscribers. Subscriber accounts can view restricted content but do not access the admin panel.

## Privacy

Cookie consent and analytics plugins (if installed) follow the site privacy policy configured by administrators.

## Getting help

Use the site **Contact** page or the email address listed in the footer.

Administrators: see [ADMIN_GUIDE.md](ADMIN_GUIDE.md).
